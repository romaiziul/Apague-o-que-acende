Comentário sobre o Projeto

O Apague o que acende é um projeto 

🎯 Objetivo do Projeto

O objetivo do projeto é criar um jogo físico baseado no jogo "White Tile? Dont tap it!", o jogo tem como objetivo trabalhar com arduino, entender a lógica por trás do código e treinar reflexos e decisões lógicas.

⚙️ Como o Sistema Funciona

Projeto de jogo interativo com Arduino, LEDs, botões e buzzer. O sistema sorteia 2 dos 4 tiles (esquema 2×2), acende os LEDs correspondentes e inicia uma contagem regressiva. O jogador deve pressionar exatamente os 2 botões acesos. Acertar ambos avança de fase; errar ou demorar termina o jogo.

🧩 Componentes do Projeto

Conforme listado no seu Lean Canvas:
